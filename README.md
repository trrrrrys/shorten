# shorten
generate url shortener using bitly api

# install
``` sh
$ go get -u github.com/trrrrrys/shorten
$ cd shorten
$ go instlll
```

# usage

``` sh
$ shorten https://github.com/trrrrrys/shorten
http://bit.ly/2GgoUYY
```

## copy on clipboard
```
$ shorten https://github.com/trrrrrys/shorten | pbcopy
```
Note: only macOS
